検証を一画面でPBR / HDRI、UV、Skin / Animation、Issue解析まで、制作中に必要な確認項目を一つの画面にまとめています。

はじめ方

まずは、この3ステップ。

読み込み、見た目の確認、原因の確認までを同じ画面で進めます。

読み込み
1

モデルを読み込む

GLB、glTF、FBXを画面へドラッグ＆ドロップします。［ファイル］から選択することもできます。

確認
2

見た目と動きを見る

向き、スケール、PBR材質、HDRI反射、ガラス、UV、Skin、Animationを切り替えて確認します。

原因を追う
3

Issueを確認する

Issue一覧から項目を選ぶと、インスペクターに対象箇所、原因、修正の手がかりが表示されます。

確認できる内容

制作中に確認できること。

見た目だけでなく、構造上の問題や納品前の確認もまとめて扱います。

VIEW

表示確認

モデルの向き、中心、スケール、欠け、裏返り、描画崩れを確認します。

Viewport / Bounds / Camera
PBR

材質・テクスチャ確認

BaseColor、Normal、Metallic、Roughness、Occlusion、Emissiveを確認します。

Material / Texture / Alpha
HDR

反射・ガラス確認

HDRI環境で金属、粗さ、透明材質の反射を同じ条件で比較します。

HDRI / IBL / Glass
MESH

メッシュ・UV確認

頂点、三角形、法線、接線、UV、マテリアル境界を追えます。

Mesh / Normal / Tangent / UV
ANIM

Skin・Animation確認

ボーンだけが動く状態やメッシュ崩れを、Clip再生とスクラブで確認します。

Skeleton / Skin / Clip
QA

Issue・レポート

問題を重要度別に整理し、確認結果をレポートとして保存できます。

Validation / Score / Report
