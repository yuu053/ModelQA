技術合格・未署名Portable

ModelQA 1.2.0

オフラインで使える Windows 向け3Dモデル検証ツールです。配布物は ModelQA.exe / ModelQA.pak / README.html の3項目だけです。

製品バージョンと配布資格

バージョンは 1.2.0、資格ラベルは 技術合格・未署名Portable です。TechnicalUnsigned は技術検証用であり、公開署名版ではありません。

対応環境

Windows 10/11 x64、DirectX 12対応GPU、1366×768以上の画面を対象にします。WARPは診断専用です。

対応形式

GLB、glTF、FBXを読み込みます。編集した名前はglTF/GLBへ書き出せます。FBXは原本を変更せずsidecarへ保存します。

基本操作

読み込み・確認

モデルを開き、Viewport、Material/Texture、UV、Mesh、Skin/Animation、Issueを確認します。

保存

既定では別名保存を使います。上書き保存は確認後のみ行い、glTF/GLBは同じフォルダーの一時ファイル完成後に原子的に置換します。

キーボードショートカット

Ctrl+O 開く　Ctrl+R レポート　Ctrl+Shift+L レイアウト初期化　Ctrl+S 上書き保存　Ctrl+Shift+S 別名保存　Esc 読み込み取消　F1 ヘルプ　Alt+F4 終了

読み込み中は保存できません。メニュー、ボタン、ショートカットは同じ可用性判定を使い、表示中の旧モデルを書き換えません。

隔離Import

大きい・危険な入力は制限付きWorkerで検査し、失敗や取消時は前に検証済みのモデルを保持します。Workerにネットワーク能力はありません。

入力上限

Portable既定のファイル、メッシュ、画像、JSON深度、文字列、メモリ上限を超える入力は、完全な読み込み前に拒否します。巨大モデルは軽量Proxyで確認できます。512MBを超えるFBXの初期表示では、形状を最大60万三角形へ縮約し、表示用法線を再生成します。

既知の制限

巨大FBXのProxyは表示確認用です。元データどおりの法線、Skin、Animation、Texture検査が必要な場合は、制作DCCから軽量化したFBXまたはGLBを書き出してください。FBXの直接上書き、未署名候補の公開配布、WARPによる実GPU資格、対応外形式の完全編集は対象外です。

設定・キャッシュ・ログ・レポート

可変データは %LOCALAPPDATA%\ModelQA に保存します。作成不能時は %TEMP%\ModelQA に安全に退避します。配布フォルダーには書き込みません。

プライバシーとネットワーク

テレメトリ、利用状況送信、自動更新、実行時ネットワーク要求は行いません。すべての検証、ログ、レポートはローカルで処理します。

ライセンスとSBOM

第三者通知、依存関係の来歴、CycloneDX SBOMは ModelQA.pak/legal/ 内の THIRD_PARTY_NOTICES.md、dependency_provenance.json、component_sbom.cdx.json にあります。

署名の確認

公開署名版では、エクスプローラーのデジタル署名または Get-AuthenticodeSignature ModelQA.exe で署名者とタイムスタンプを確認してください。TechnicalUnsignedは署名済みと表示しません。

サポート

モデル、再現手順、ModelQAのバージョン、Issue/レポートを添えて配布元のサポート窓口へ連絡してください。秘密情報や不要な原本を送信しないでください。
